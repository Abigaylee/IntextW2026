# ML pipelines (IS 455)

Put **all** team ML pipeline notebooks in this folder so submission stays organized.

## Start here

- **`IS455_Showcase_Example_Pipeline.ipynb`** — Inlines roadmap/backlog-style guidance **plus** one full example pipeline (*education_records*) with sample rubric prose and runnable cells. Open this first to see the “amazing” pattern.
- **`blank_notebook.ipynb`** — Template: set `PIPELINE_NAME` and fill in your narrative.
- **`ALL_PIPELINES_ROADMAP.md`** — Full Tier A/B table, definition of done, suggested order.
- **`pipeline_backlog.md`** — Business questions per problem area.
- **`pipeline_kit.py`** — Shared explanatory + predictive models and optional `artifacts/` export.
- **`run_all_pipelines.ipynb`** — Run every registered pipeline once.

Working directory for notebooks should be **`ml-pipelines/`** so paths to `../datasets/` resolve correctly.
